Ionic implements view caching which means: * The Angular controller is not constructed each time * The controller's $scope is detached from the angular lifecyle when it is cached and not active The main change is in Angular controllers you will put logic into the Ionic events, such as $ionicView.beforeEnter. You will see many examples in the codebase.

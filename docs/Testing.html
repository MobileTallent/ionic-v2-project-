[Index](index.html) To run the unit tests at the command line run **karma start karma.conf.js** (This will run the test continuously each time you make a change) To run the Parse integration tests at the command line run **karma start karma-parse.conf.js** To start the Protractor end-to-end tests which run in the browser in one command shell/prompt run: **webdriver-manager start** Then in another command shell/prompt run **protractor protractor.config.js** Links: https://www.airpair.com/angularjs/posts/testing-angular-with-karma http://developer.mobilecaddy.net/2015/05/announcing-protractor-ionic-locator/
